1 Ordenamiento Burbuja Simple ▼

El ordenamiento burbuja recorre el arreglo repetidamente, comparando elementos adyacentes e intercambiándolos si están en el orden incorrecto. En cada pasada, el elemento más grande “burbujea” hasta su posición final. Su complejidad es O(n²) en el peor caso.

        for (int i = 0; i < n - 1; i++) {

                if (vec[j] > vec[j + 1]) {

                    vec[j] = vec[j + 1];

2 Burbuja Mejorada ▼

La burbuja mejorada agrega una bandera (flag) para detectar si ocurrió algún intercambio en una pasada. Si no hubo intercambios, el arreglo ya está ordenado y el algoritmo termina antes, optimizando el caso promedio.

3 QuickSort ▼

QuickSort es un algoritmo divide y vencerás. Selecciona un pivote, particiona el arreglo colocando los elementos menores al pivote a la izquierda y los mayores a la derecha, y luego ordena recursivamente cada subarreglo. Su complejidad promedio es O(n log n).

import java.util.*;
import java.lang.*;
import java.io.*;

class Codechef
{
    public static void mostrar(int[] vec) {
        for (int num : vec) {
            System.out.print(num + " ");
        }
        System.out.println();
    }

    public static void quickSort(int[] vec, int low, int high) {
        if (low < high) {
            int pi = partition(vec, low, high);
            quickSort(vec, low, pi - 1);
            quickSort(vec, pi + 1, high);
        }
    }

    public static int partition(int[] vec, int low, int high) {
        int pivot = vec[high];
        int i = low - 1;
        for (int j = low; j < high; j++) {
            if (vec[j] < pivot) {
                i++;
                int temp = vec[i];
                vec[i] = vec[j];
                vec[j] = temp;
            }
        }
        int temp = vec[i + 1];
        vec[i + 1] = vec[high];
        vec[high] = temp;
        return i + 1;
    }

    public static void main(String[] args) throws java.lang.Exception {
        int[] vec = {45, 78, 12, 89, 33, 56, 71, 9, 64, 27};

        System.out.println("Arreglo original:");
        mostrar(vec);

        quickSort(vec, 0, vec.length - 1);

        System.out.println("Arreglo ordenado:");
        mostrar(vec);
    }
}

4 ShellSort ▼

ShellSort es una optimización del ordenamiento por inserción. Compara elementos separados por un gap que se reduce progresivamente. Esto permite que elementos lejanos se muevan rápidamente a su posición, reduciendo el número total de intercambios.

import java.util.*;
import java.lang.*;
import java.io.*;

class Codechef
{
    public static void mostrar(int[] vec) {
        for (int num : vec) {
            System.out.print(num + " ");
        }
        System.out.println();
    }

    public static void shellSort(int[] vec) {
        int n = vec.length;
        for (int gap = n / 2; gap > 0; gap /= 2) {
            for (int i = gap; i < n; i++) {
                int temp = vec[i];
                int j = i;
                while (j >= gap && vec[j - gap] > temp) {
                    vec[j] = vec[j - gap];
                    j -= gap;
                }
                vec[j] = temp;
            }
        }
    }

    public static void main(String[] args) throws java.lang.Exception {
        int[] vec = {45, 78, 12, 89, 33, 56, 71, 9, 64, 27};

        System.out.println("Arreglo original:");
        mostrar(vec);

        shellSort(vec);

        System.out.println("Arreglo ordenado:");
        mostrar(vec);
    }
}

5 Selección Directa ▼

El ordenamiento por selección divide el arreglo en dos partes: ordenada y desordenada. En cada paso, busca el menor elemento en la parte desordenada y lo intercambia con el primer elemento de esa parte. Su complejidad es O(n²).

    }

        for (int i = 0; i < n - 1; i++) {

            }

6 Inserción Simple ▼

El ordenamiento por inserción construye la lista ordenada de izquierda a derecha, tomando un elemento no ordenado e insertándolo en la posición correcta dentro de la parte ya ordenada. Es eficiente para arreglos pequeños o casi ordenados.

7 Ordenación Shaker ▼

Shaker Sort (o cocktail sort) es una variante del burbuja que recorre el arreglo en ambas direcciones. En cada iteración, primero desplaza el elemento más grande al final y luego el más pequeño al inicio, reduciendo el rango de búsqueda por ambos extremos.

Presiona "Comenzar" para iniciar la animación.

import java.util.*;
import java.lang.*;
import java.io.*;

class Codechef
{
    public static void mostrar(int[] vec) {
        for (int num : vec) {
            System.out.print(num + " ");
        }
        System.out.println();
    }

    public static void shakerSort(int[] vec) {
        int n = vec.length;
        int left = 0;
        int right = n - 1;
        while (left < right) {
            for (int i = left; i < right; i++) {
                if (vec[i] > vec[i + 1]) {
                    int temp = vec[i];
                    vec[i] = vec[i + 1];
                    vec[i + 1] = temp;
                }
            }
            right--;
            for (int i = right; i > left; i--) {
                if (vec[i] < vec[i - 1]) {
                    int temp = vec[i];
                    vec[i] = vec[i - 1];
                    vec[i - 1] = temp;
                }
            }
            left++;
        }
    }

    public static void main(String[] args) throws java.lang.Exception {
        int[] vec = {45, 78, 12, 89, 33, 56, 71, 9, 64, 27};

        System.out.println("Arreglo original:");
        mostrar(vec);

        shakerSort(vec);

        System.out.println("Arreglo ordenado:");
        mostrar(vec);
    }
}

8 MergeSort ▼

MergeSort es un algoritmo divide y vencerás que divide el arreglo en mitades, las ordena recursivamente y luego las fusiona (merge) de forma ordenada. Su complejidad es O(n log n) en todos los casos, pero requiere memoria adicional.

Visualizador
Presiona "Comenzar" para iniciar la animación.
Código Fuente
import java.util.*;
import java.lang.*;
import java.io.*;

class Codechef
{
    public static void mostrar(int[] vec) {
        for (int num : vec) {
            System.out.print(num + " ");
        }
        System.out.println();
    }

    public static void mergeSort(int[] vec, int left, int right) {
        if (left < right) {
            int mid = (left + right) / 2;
            mergeSort(vec, left, mid);
            mergeSort(vec, mid + 1, right);
            merge(vec, left, mid, right);
        }
    }

    public static void merge(int[] vec, int left, int mid, int right) {
        int n1 = mid - left + 1;
        int n2 = right - mid;

9 HeapSort — Montículo ▼

HeapSort utiliza una estructura de montículo binario (heap) para ordenar. Primero construye un max-heap, luego extrae repetidamente la raíz (el máximo) y la coloca al final, reduciendo el tamaño del heap. Su complejidad es O(n log n).

        int n = vec.length;
        for (int i = n / 2 - 1; i >= 0; i--)

            vec[largest] = temp;

10 BucketSort — Cubo ▼

BucketSort distribuye los elementos en varios cubos (buckets), ordena cada cubo individualmente y luego concatena los cubos. Es eficiente cuando los datos están distribuidos uniformemente.

11 CountingSort — Conteo ▼

CountingSort cuenta la frecuencia de cada valor único y usa esa información para colocar cada elemento en su posición correcta. Es un algoritmo de ordenamiento no comparativo con complejidad O(n + k), donde k es el rango de valores.

        int n = vec.length;
        int max = vec[0], min = vec[0];

            if (vec[i] < min) min = vec[i];

class ListaCircularDoble {
    Nodo cabeza;
    public ListaCircularDoble() {
        cabeza = null;
    }
    public boolean estaVacio() {
        return cabeza == null;
    }
    public void insertar(int dato) {
        Nodo nuevo = new Nodo(dato);
        if(estaVacio()) {
            nuevo.siguiente = nuevo;
            nuevo.anterior = nuevo;
        } else {
            Nodo despues = cabeza.siguiente;
            nuevo.siguiente = despues;
            nuevo.anterior = cabeza;
            cabeza.siguiente = nuevo;
            despues.anterior = nuevo;
        }
        cabeza = nuevo;
    }
    public void mostrar() {
        if(estaVacio()) {
            System.out.println("Lista esta vacia");
            return;
        }
        Nodo actual = cabeza;
        do {
            System.out.print(actual.dato + " ");
            actual = actual.siguiente;
        } while(actual != cabeza);
        System.out.println(" ");
    }
    public void mostrarReversa() {
        if(estaVacio()) {
            System.out.println("Lista esta vacia");
            return;
        }
        Nodo actual = cabeza.anterior;
        do {
            System.out.print(actual.dato + " ");
            actual = actual.anterior;
        } while(actual != cabeza.anterior);
        System.out.println(" ");
    }

        if(estaVacio()) return;
        if(cabeza.siguiente == cabeza) {
            cabeza = null;
        } else {
            Nodo elim = cabeza.siguiente;

19 T. Pila — LIFO (Last In, First Out) ▼
