Recopilación de los ejercicios vistos en la clase de **Estructura de Datos**, llevados a una página web interactiva con la ayuda de **vibe coding**.

## Características

- Cada ejercicio incluye una **breve explicación teórica**, un **simulador visual interactivo** y el **código fuente** en **Java**, **Python** y **C++**.
- Los simuladores de ordenamiento permiten ajustar cantidad de elementos y velocidad de la animación.
- Las listas enlazadas, pila y cola tienen paneles interactivos para probar las operaciones en tiempo real.
- Interfaz con tema oscuro/claro.
- Todo el contenido está en un solo archivo HTML.
